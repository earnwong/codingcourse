Arrays

Arrays are used to store lists of data. You can access, add, or remove items from an array. 


let fruits = ["Apple", "Banana", "Orange"];
console.log(fruits[0]); // Output: Apple
fruits.push("Mango");   // Add an item
console.log(fruits);    // Output: ["Apple", "Banana", "Orange", "Mango"]

Objects

Objects are used to store key-value pairs. They are great for representing real-world entities. 


let person = { 
  name: "John",
  age: 15,
  isStudent: true
};
console.log(person.name); // Output: John
person.age = 16;         // Update a property
console.log(person.age); // Output: 16

Loops

Loops allow you to repeat a block of code multiple times. The most common loops are for and while. 


// For loop
for (let i = 0; i < fruits.length; i++) {
  console.log(fruits[i]);
}

// While loop
let count = 0;
while (count < 3) {
  console.log("Count: " + count);
  count++;
}

Functions

Functions are reusable blocks of code that perform a specific task. They can take inputs (parameters) and return outputs. 


// Function declaration
function greet(name) {
  return "Hello, " + name + "!";
}
console.log(greet("John")); // Output: Hello, John!

// Arrow function (modern syntax)
const add = (a, b) => a + b;
console.log(add(2, 3)); // Output: 5
