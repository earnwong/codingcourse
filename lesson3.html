DOM Manipulation

The Document Object Model (DOM) is a structured representation of an HTML document. The browser creates this structure automatically when a webpage loads. Think of the DOM as a tree where each HTML element is a node in this tree.

What is a Tree?

A tree is a way to organize information in a hierarchical structure. In a tree:

  • Nodes: Each item in the tree is called a node. For example, in the DOM, every HTML element (like <div>, <p>, or <button>) is a node.
  • Parent and Child: Nodes can have relationships. A parent node is one that contains other nodes, and a child node is one that is contained within another node. For example, in the HTML below: 
    <div>
  <p>This is a paragraph.</p>
  <button>Click Me</button>
</div>
    The <div> is the parent of the <p> and <button>, and the <p> and <button> are children of the <div>.
  • Root: The topmost node in the tree is called the root. In the DOM, the <html> element is the root.

Parent-Child Relationships in the DOM

Example: Shopping List

A simple example is a shopping list, where:

  • The <ul> (unordered list) is the parent.
  • Each <li> (list item) inside it is a child.
  • We can add and remove children dynamically using JavaScript.

Key Methods

  • parent.appendChild(child) → Adds a child element to a parent.
  • parent.removeChild(child) → Removes a child element from a parent.

How It Works

When you add an item to the shopping list, a new <li> is created and appended to the <ul>. Clicking "Remove" will remove that <li> from the parent.

This concept is essential for interactive web pages, allowing dynamic content updates, to-do lists, form validation, and more!

The DOM Tree

The DOM tree is a hierarchical structure that represents the HTML document. Each element in the HTML document is a node in this tree. Here's an example of what a DOM tree might look like: 

Document
├── html
│   ├── head
│   ├── body
│       ├── h1
│       ├── p
│       ├── div (container)
│           ├── p (inside div)
│           ├── button

Each of these elements can be accessed, changed, or removed using JavaScript. This allows you to dynamically update the content, style, or structure of a webpage.

How to Access Elements

You can use JavaScript to access and modify elements in the DOM using various methods. Here are some common methods:

  • document.getElementById("id") - Selects an element by its unique ID.
  • document.querySelector("selector") - Selects the first element that matches a CSS selector.
  • document.createElement("tag") - Creates a new HTML element.
  • document.getElementsByClassName("class") - Selects all elements with a specific class name.
  • document.getElementsByTagName("tag") - Selects all elements with a specific tag name.

These methods allow you to find and manipulate elements in the DOM. For example, you can change the text inside a paragraph, add a new button, or change the color of a heading.

Example: DOM Manipulation

This is some example text.

HTML


<p id="example-text">This is some example text.</p>
<button onclick="changeText()">Change Text</button>
<button onclick="changeColor()">Change Color</button>
<button onclick="addParagraph()">Add Paragraph</button>

JavaScript


function changeText() {
  document.getElementById("example-text").textContent = "The text has been changed!";
}

function changeColor() {
  document.getElementById("example-text").style.color = "red";
}

function addParagraph() {
  let newParagraph = document.createElement("p");
  newParagraph.textContent = "This is a new paragraph!";
  document.querySelector(".example").appendChild(newParagraph);
}

Event Handling

Events are actions that happen in the browser, such as clicks, mouse movements, or keyboard input. You can use JavaScript to respond to these events.

Example: Event Handling

HTML


<button id="alert-button">Click Me</button>
<button id="color-button">Change Background Color</button>

JavaScript


document.getElementById("alert-button").addEventListener("click", function () {
  alert("Button clicked!");
});

document.getElementById("color-button").addEventListener("click", function () {
  document.body.style.backgroundColor = "lightblue";
});

Form Validation

Form validation ensures that user input is correct before submitting it. You can use JavaScript to check for errors and display messages.

Example: Form Validation

HTML


<form id="example-form">
  <input type="text" id="example-input" placeholder="Enter something">
  <button type="submit">Submit</button>
</form>

JavaScript


document.getElementById("example-form").addEventListener("submit", function (event) {
  let input = document.getElementById("example-input").value;
  if (input.trim() === "") {
    alert("Please enter a value!");
    event.preventDefault(); // Prevent form submission
  }
});

  • Add tasks to a list.
